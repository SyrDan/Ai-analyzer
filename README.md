# AI Учебный Помощник

Веб-приложение на Python с машинным обучением для анализа учебных текстов и генерации персональных рекомендаций по изучению материала.

## Возможности

- **Анализ текста**: Определение сложности, объема и структуры материала
- **ML-классификация**: Автоматическое определение предмета/темы текста
- **Извлечение ключевых слов**: Выделение важнейших терминов с помощью TF-IDF
- **Оценка времени**: Расчет необходимого времени на изучение
- **Персональные рекомендации**: AI-генерация советов по методам изучения
- **План обучения**: Пошаговый план для эффективного усвоения материала
- **Красивый интерфейс**: Современный, адаптивный дизайн

## Требования

- Python 3.8+
- pip (менеджер пакетов Python)

##Установка

### 1. Клонируйте или скачайте проект

```bash
# Если используете git
git clone <repository-url>
cd ai-study-helper

# Или просто скачайте и распакуйте архив
```

### 2. Создайте виртуальное окружение (рекомендуется)

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите приложение

```bash
python app.py
```

Приложение будет доступно по адресу: **http://localhost:5000**

## Как использовать

1. **Откройте приложение** в браузере (http://localhost:5000)
2. **Вставьте текст** для анализа (конспект, статью, главу из учебника)
3. **Нажмите "Анализировать текст"**
4. **Получите результаты**:
   - Статистика по тексту
   - Уровень сложности
   - Ключевые слова и термины
   - Анализ структуры
   - Персональные рекомендации
   - План изучения

## Технологии и ML-алгоритмы

### Backend
- **Flask**: Веб-фреймворк
- **scikit-learn**: Машинное обучение
- **NLTK**: Обработка естественного языка
- **NumPy**: Численные вычисления

### ML-компоненты
- **TF-IDF Vectorizer**: Извлечение ключевых слов
- **Tokenization**: Разбиение текста на предложения и слова
- **Stop Words Filtering**: Удаление незначимых слов
- **Complexity Analysis**: Расчет индекса сложности текста
- **Subject Classification**: Определение темы по ключевым словам

### Frontend
- **HTML5/CSS3**: Структура и стили
- **JavaScript (ES6+)**: Интерактивность
- **Fetch API**: Асинхронные запросы

## Настройка

### Изменение порта
В файле `app.py`:
```python

```

```python
}
```

## Решение проблем

### Ошибка при установке NLTK
```bash
# Вручную загрузите данные NLTK
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords')"
```

### Порт уже занят
```bash
# Измените порт в app.py или освободите 5000
# Windows: netstat -ano | findstr :5000
# Linux/Mac: lsof -ti:5000
```

### Проблемы с кодировкой
Убедитесь, что файлы сохранены в UTF-8 кодировке.
